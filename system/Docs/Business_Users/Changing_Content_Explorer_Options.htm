Changing Content Explorer Options

You may change the colors and font of the Content Explorer for the current session.

To change Content Explorer options:

  1. In the Menu Bar, Choose View > Options.

    Rhythmyx displays the Display Options dialog.

  2. Optionally, change the Background Color, the Main Display Colors, or the Menu Text Color.
    1. Click after the option.

      Rhythmyx displays a Choose Color dialog.

    2. Use any of the tabs to choose a color.
    3. Click [OK].
  3. Optionally, change the Main Display or Menu font type or size by choosing new options in the drop lists beside Font.
  4. Optionally, change the Main Display or Menu font style by checking Bold or Italic.
  5. To use the display options set for the operating system, check Use OS Settings.
  6. Click one of the buttons at the bottom of the dialog:
    • [Default] - return to the default settings;
    • [OK] - apply your new settings and close the Display Options dialog;
    • [Cancel] - close the Display Options dialog without applying your new settings;
    • [Help] - open online help.

To view the Rhythmyx interface with accessibility display options (magnified fonts and high resolution color schemes), set up the Screen Magnifier that is shipped with Windows XP. To access the Screen Magnifier go to Accessories > Accessibility.
