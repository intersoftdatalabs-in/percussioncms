Accessing the Active Assembly Interface from a Content Editor

To access the Active Assembly interface from a Content Explorer content editor:

  1. In Content Explorer, open the Content Item in a content editor. You can only edit the item if it is checked out by you or available to you to check out.
  2. Scroll down to the Template links in the Related Content box at the bottom of the content editor.

  3. Click the name of a Template.

    Rhythmyx displays the Template in the Active Assembly interface.

For information on using the Active Assembly interface, see the online help available from the interface.
