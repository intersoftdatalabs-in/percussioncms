Acting on Related Content in a Document Slot

To act on content in a Slot in the Document Assembly window:

  1. Click the Slot in the Navigation pane to display its contents in the Display pane.
  2. Right-click the Content Item in the Display pane.

    Rhythmyx displays an Action Menu.

  3. In the Action Menu, select the action that you want to perform on the Content Item.

For more information, see Action Menus and the Working with Content Items section in general.
