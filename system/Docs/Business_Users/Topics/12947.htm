Refreshing Lifecycle Analysis Report Data

If you have saved a copy of the workbook locally and you want the latest data, you must refresh the workbook data. To refresh the data:

  1. In the Menu bar, choose Rhythmyx > Refresh Data.
  2. If you have not already entered your user name and password, Excel will display a login dialog. Enter your user name and password.
  3. MS Excel updates the workbook with the latest data retrieved from the Rhythmyx server.
