Using Linkback

Linkback is a Rhythmyx feature that allows you to link from a published HTML page to the Content Item from which that page is generated. To use linkback, you must add the linkback bookmarklets to your browser. To add linkback bookmarklets:

  1. Access the Rhythmyx Application Server Homepage.

    To access this page, start a browser and in the address field, enter http://Rhythmyx:9992 where Rhythmyx is the name or IP address of the Rhythmyx Server and 9992 is the server port. If you have a bookmark to the Rhythmyx server, you can access this page by clicking on the bookmark, and in the address field, removing all text from the single slash after the port to the end.

  2. On the Application Server Homepage, click the Linkback Help link.

    Your browser will be directed to the Linkback Help page, which includes two bookmarklets.

  3. Add the bookmarklets you want to use to your browser.

When viewing an HTML page published by Rhythmyx, you can access its Content Item by clicking on the bookmarklet.
