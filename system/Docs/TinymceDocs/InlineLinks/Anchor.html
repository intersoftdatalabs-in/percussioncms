Insert/Edit Anchor Dialog

Insert/Edit Anchor Dialog and Field Definitions

This dialog is accessed from the rich text editor when you place your cursor at a specific point in the text to insert an anchor or when you select an existing anchor and click in the rich text editor toolbar. An anchor inserted in the text enables a link in this page or another page to jump to the anchor's location (so the web visitor is taken to a specific topic or definition mentioned in the text).

Fields

Anchor name - A value for a name attribute for the anchor tag in the HTML source code for the rich text content. The dialog does not require you to enter this field, but if you do not enter an Anchor name, it is not possible for a link to access this anchor.
