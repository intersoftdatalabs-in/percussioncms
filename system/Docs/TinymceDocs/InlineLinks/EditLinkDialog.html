Insert/Edit Link Dialog and Field Definitions

If you want to insert an inline link, select text or an image in the rich text editor and click .
If you want to edit an inline link, click anywhere on the link and click

If the text or image does not already contain a link on it, the dialog fields are empty to enable you to create a link. If the text or image already contains a link on it, the dialog fields are filled with data defining the link, and you can edit them.

Fields

Link URL - Holds the URL of the page or anchor that the link accesses. You can do one of the following:
  • Enter an external URL.
  • Do not enter anything in the field and choose an anchor in the Anchors drop list. When you choose an anchor, the dialog adds the anchor's address #anchorname to this field.
  • Browse for an existing page in CM1. Use the following procedure:

    1. Click .

      2. The Select Page dialog opens:

      The box on the bottom of the page shows all of the sites in your system. In the example, the only site is MemberSite.
    2. Double click on the site that you want to browse.

      3. Now the site name appears in the Location drop list, and the contents of the first level of the site appear in the bottom box.
    3. If you double-click a folder or section folder in the box, it appears in the Location drop list and its contents appear in the bottom box..


    4. Once you have navigated to the folder holding the page that you want to use, click the page in the box.

      5. The name appears in the Selected Page field.
    5. Click [Save] to close the Select Page dialog and insert the URL of the page that you selected into the Link URL field in the Insert/Edit Link dialog..
Anchors - Drop list including all anchors on the current page and an empty option. Choose one of the anchors if you want the link to access that point on the page. When you choose an anchor, the dialog overwrites any previously entered value in Link URL with the anchor's address #anchorname.
Target - Window or frame in which page accessed by the link opens. Options are:
  • (default) Open in this window / frame
  • Open in new window (_blank)
  • Open in parent window / frame (_parent)
  • Open in top frame (replaces all frames (_top)
Title - Tooltip text. When you hover your cursor over the link, text entered into Title appears as a tooltip. Default is the title of the page accessed by the link.
