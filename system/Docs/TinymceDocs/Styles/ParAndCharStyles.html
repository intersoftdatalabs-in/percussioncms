Paragraph and Character Styles

In the rich text editor, you can place an inline link on text or an image that you have selected. When a site visitor clicks the text or image that contains the link, the external URL or internal page that the link accesses opens. The rich text editor also gives you the ability to insert a link that jumps to a specific point in a page (for example, a title or a definition). To do this, insert an anchor at the point in the page that you want the link to access, and then insert the link that accesses the anchor over text or an image. In the following graphic, the contributor has inserted an anchor before the words Find a Provider, and then has placed a link on the text search for a provider. Now the link search for a provider will take the site visitor to the words Find a Provider, where the anchor appears. You can also insert a link over an image. On the page below, the contributor has placed a link over the telephone graphic. Now, if the user clicks the graphic, the Contact Us page opens.



Button Function
Opens a dialog that enables you to configure and insert an inline link or edit an existing inline link. The link can access an external URL, an internal page, or a page anchor.
Removes an inline link. To remove the link, position your cursor within the link text or select the linked image, and then click this button.
Opens a dialog that lets you insert or edit an anchor within the text.
