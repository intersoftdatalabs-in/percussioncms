HTML Mode

Button Function
Toggles between rich text mode and HTML mode.
The rich text editor is a WYSIWIG editor that formats HTML code. Use the button to toggle between rich rext mode and HTML mode, which enables you to correct your HTML markup or to work in HTML.

To use HTML mode:

  1. Click to switch to HTML mode.
    The editor looks the same, but many of the buttons are disabled. Any content that you have entered as rich text now appears as HTML markup.
  2. Modify or view the HTML text, or proceed to work in HTML.
  3. When you are done, either click [SAVE] to save your changes and return to the page in the Content tab (the rich text editor closes in both rich text mode and HTML mode), or click to return to rich text mode.
    Note that when you when you click [SAVE] or , the HTML editor may reformat markup into notations that it prefers and attempts to remove invalid markup. After you save your content in either mode, the editor inserts <div> tags around your HTML markup unless they are already present.
