Table Insert/Modify Dialog


   

Note: Definitions of General and Advanced tab fields follow standard HTML definitions.

General Tab Fields

Cols - The number of columns in the table.
Rows - The number of rows in the table.
Cellpadding - The space between the border and the text in each cell.
Cellspacing - Width of borders between cells.
Alignment - Position of table on the page.
Border - Size of external border.
Width - Width of table in pixels.
Height - Height of table in pixels.
Class - Drop list that allows you to enter a CSS style for formatting the table. To add a CSS style, choose (value) in the drop list, and enter its name.
Table Caption - Checkbox that enables you to enter a caption above the table.

Advanced Tab Fields

Id - Table identifier for links or css fields.
Summary - Table attribute that describes table's function and appearance for screen readers.
Style - Table attribute that sets a style for the entire table. Height entered in General tab is automatically entered as a style.
Language code - Standard language abbreviation for parser to use.
Background image - Opens a dialog that enables you to upload or select a background image to insert in the table.
Frame - Drop list of options for framing table cells.
Rules - Drop list of options for adding rule lines to the table.
Language direction - Enables you to select Left to right or Right to Left for language direction in table. Default is Left to right.
Border color - Enables you to select a custom border color.
Background color - Enables you to select a custom color for all cells in the table.
