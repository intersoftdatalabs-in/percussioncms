Table Cell Properties Dialog

   

General Tab Fields

Alignment - Horizontal alignment of text in cell.
Cell Type - Drop list that lets you choose if the cell is treated as a header cell or a data cell in the HTML markup.
Vertical Alignment - Vertical alignment of text in cell.
Scope - Drop list indicating which row or column cells to apply these edits to.
Width - Width of cell in pixels.
Height Height of cell in pixels.
Class - Drop list that allows you to enter a CSS style for formatting the cell. To add a CSS style, choose (value) in the drop list, and enter its name.
Update drop list - Drop list of options indicating portion of table to apply the dialog's edits to. Choices are cell, row, or table.

Advanced Tab Fields

Id - Cell identifier for links or css fields.
Style - Attribute that sets a style for the cell. Height entered in General tab is automatically entered as a style.
Language direction - Enables you to select Left to right or Right to Left for language direction in cell. Default is Left to right.
Language code - Standard language abbreviation for parser to use.
Background image - Enables you to upload or select a background image to insert in the row.
Border color - Enables you to select a custom border color for the cell.
Background color - Enables you to select a custom color for all cells in the row.
Update drop list - Drop list of options indicating portion of table to apply the dialog's edits to. Choices are cell, row, or table.
