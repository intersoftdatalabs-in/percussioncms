Table Row Properties Dialog

   

General Tab Fields

Row in table part - Drop list that lets you choose if the the row is treated as a header row, body row, or footer row in the HTML markup.
Alignment - Horizontal alignment of text in cells in this row.
Vertical alignment - Vertical alignment of text in cells in this row.
Class - Drop list that allows you to enter a CSS style for formatting the row. To add a CSS style, choose (value) in the drop list, and enter its name.
Height - Height of row in pixels.
Update drop list - Drop list that lets you choose which rows in table to update with the properties entered in the dialog.

Advanced Tab Fields

Id - Row identifier for links or css fields.
Style - Attribute that sets a style for the row. Height entered in General tab is automatically entered as a style.
Language direction - Enables you to select Left to right or Right to Left for language direction in row. Default is Left to right.
Language code - Standard language abbreviation for parser to use.
Background image - Enables you to upload or select a background image to insert in the row.
Background color - Enables you to select a custom color for all cells in the row.
Update drop list - Drop list that lets you choose which rows in table to update with the properties entered in the dialog.
