How to Access the Server Administrator

You can access the Server Administrator directly if it is installed on your machine, or remotely through a browser.  For you to have direct access, the Server Administrator must be installed on your machine.

To access Administration directly:

  • Double-click the Rhythmyx Server Administrator icon OR
  • Run RhythmyxServerAdministrator.exe

To access Administration from a remote machine:

  • Enter the URL in this format: http://<Rhythmyx server>:<port number>/Rhythmyx/Administration/Admin.html where Rhythmyx Server is the name of the machine where Rhythmyx resides and port number is the port where the Rhythmyx Server listens for requests.

The Server Administrator window displays the title:
Rhythmyx Server Administrator for <Rhythmyx server>:<port>
