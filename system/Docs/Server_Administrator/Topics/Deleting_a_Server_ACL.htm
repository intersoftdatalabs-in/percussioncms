Deleting a Server ACL

Be careful when clicking [Delete].  When you click [Delete] Rhythmyx deletes the highlighted entry without displaying a warning first.

To delete a server ACL:

  1. Click the Security tab.
  2. Click the Server ACL tab.
  3. Click the Server ACL you wish to delete.
  4. Click the [Delete] button.  Rhythmyx deletes the Server ACL.
