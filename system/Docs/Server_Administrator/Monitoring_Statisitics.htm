Monitoring Statistics

You can view and monitor performance statistics for many sectors of activity. Rhythmyx provides these options:

  • Viewing server performance
  • Viewing application performance

By issuing commands to the server, performance information can be gathered. Refer to the Interfacing by Remote Console section for information on commands including, show status application and show status server.
