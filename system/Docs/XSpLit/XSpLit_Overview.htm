XSpLit Overview

XSpLit works "behind-the-scenes" with Rhythmyx. XSpLit creates XSL stylesheet, XML document, and XML Document Type Definition (DTD) files from a specially marked up HTML file. Labeling static content in the HTML file specifies where to place dynamic data, and XSpLit generates XSL, XML, and DTD files based on these labels.

This version of XSpLit is stand alone so you will not be able to work with databases. You can do the following:

  • Generate an XSL style sheet to produce the format of your original document.
  • Generate a Document Type Definition (DTD) file based on the names used in labeling.
  • Generate and modify an XML sample document using the labeled static content as sample data.
  • Generate a new HTML document from the XSL style sheet and XML file.
