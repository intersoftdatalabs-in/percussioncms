Formatting Repeated Fields

When you create an XML document from back-end data, you may need to repeat the same field in that document multiple times, each time with a different value. To format these repeated fields, XSpLit generates style sheets capable of repeating both the fields and the surrounding HTML formatting tags.

You can also nest repeated fields and formatting within other repeating fields and formatting. For example, you could nest a repeating bulleted list of items within a repeating row of a table.

You control repeating formatting for fields based on:

  • Where you place the fields in an HTML page
  • The hierarchical node names shared by the repeated fields
  • Specification of the repeating context

You can repeat the following elements:

  • table rows <tr>
  • lists <li>
  • select forms <select>
  • select <option>
