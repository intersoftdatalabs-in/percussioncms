Hierarchical Field Names

Hierarchical field names are names grouped together under a common node name in a tree structure, such as the following example:

Names begin with the nodes and progress from left to right to the specific field. Format hierarchical names using slashes. For example:

Employee/Personal/Name/First

Employee/Personal/Name/Last

Employee/Personal/HomePhone

Employee/Company/Position/Department

Employee/Company/WorkPhone

If you want a field to repeat several times in a document, with different values each time, some level of hierarchical naming is usually required.
