Exposed Syntax-Fields as Tag Attributes

To use Exposed Syntax for a tag attribute, specify the field name as the value of the attribute:

<tagname attribute="psx-fieldname">

Examples:

<a href="psx-Person/Link">click here</a>

<img src="psx-Photo/Name" alt="psx-Photo/Description">

<font size="psx-Order/Pointsize">Hello world.</font>
