The data cataloger provides metadata about back-end drivers, tables, etc. It follows the specification defined by the cataloger for all catalog requests. Data catalog requests are handled internally by the Data Manager

The Data Manager supports the following request types, and their corresponding handlers:

Request TypeHandler
Driver PSDriverCatalogHandler
DriverSupport PSDriverSupportCatalogHandler
Server PSServerCatalogHandler
Database PSDatabaseCatalogHandler
Schema PSSchemaCatalogHandler
TableTypes PSTableTypesCatalogHandler
Table PSTableCatalogHandler
Column PSColumnCatalogHandler
UniqueKey PSUniqueKeyCatalogHandler
ForeignKey PSForeignKeyCatalogHandler
Index PSIndexCatalogHandler
