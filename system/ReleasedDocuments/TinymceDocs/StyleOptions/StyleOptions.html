Style Options

You can use the Style drop list to add CSS style classes to your text. You can add the style classes to HTML tags or add them to your text inline.

Adding Style Classes to HTML Tags

The rich text editor displays the element path for each HTML tag at the bottom of the rich text editor:

To add style classes to a tag, select the in the element path (for example, the <p> tag in the illustration above), then click on the Style drop list and select the CSS class you want to apply to the tag. See below for a list of available classes.

The example code below shows the Body text 1 style CSS class added to an example paragraph. Applying this style adds the class perc-body-text1 to the <p> tag.

<p class="perc-body-text-1">Simply fill out the form below and we'll do the rest! We'll match the free volunteer travel opportunity with the selections you make and send you an email update.  You're under no obligation to pursue the opportunity, and may remove yourself from our list at any time.</p>

The specific formatting of the class is defined in the theme CSS.

Adding Style Classes Inline

To add style classes to text inline, select the text to which you want to apply the style class, then click on the Style drop list and select the CSS class you want to apply to the tag. See below for a list of available classes.

CM1 adds a <span> tag around the selected text. The selected CSS class is added to the <span> tag. In the example below, the style CSS class Quote 1 has been added to the second sentence of the text.

<p class="perc-body-text-1">Simply fill out the form below and we'll do the rest! <span class="perc-quote-1">We'll match the free volunteer travel opportunity with the selections you make and send you an email update .</span>   You're under no obligation to pursue the opportunity, and may remove yourself from our list at any time.</p>

Removing Inline Style Classes

To remove inline style classes, move the cursor into the text contained within the <span> and click the Style Eraser.

NOTE: If you click the Style Eraser while you have an HTML tag selected in the element path, all inline style classes will be removed from the text within that tag.
