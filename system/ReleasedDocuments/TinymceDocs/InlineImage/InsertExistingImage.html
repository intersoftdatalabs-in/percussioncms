Insert Existing Image

You insert an existing image into a rich text editor by selecting an image that is already stored in CMS as an asset. Currently, you can only insert existing images stored as assets in CM1 into the rich text editor.

To insert an existing image into a rich text editor:

  1. Open a rich text editor widget for edit.
  2. Click in the location in the text where you want to insert the image.
  3. Click .
    The Insert/Edit Image dialog opens.
  4. In the General tab, click [Use Existing Image].
    The Select Image dialog opens.

    The box at the bottom of the dialog lists the Assets root node that appears in the Finder.
  5. Double-click Assets and navigate through the folder structure until you find an image asset that you want to select.
    If you want to move back up the folder structure, select a folder in the Location drop list.
    6. Images are stored either in the default Assets/uploads/images folder or in user-created folders.
  6. Click the image name that you want to insert into the rich text.
    The name appears in the Image Name field.

  7. Click [Save] to return to the Insert/Edit Image dialog. The fields are now filled with information from the selected image, and the Preview box displays the image.

    (If you do not want to insert an image, click [Cancel] in the Select Image dialog to return to the Insert/Edit Image dialog. The fields in the Insert/Edit Image dialog remain unfilled.)
  8. Once the Insert/Edit Image dialog displays the image and its properties, you can change the Image description and the Title fields. To create a thumbnail image in Image size, click the Thumbnail Image radio button. The Preview box now displays the thumbnail image.
    If you want to change the default appearance of the image, go to the Insert/Edit Image dialog.
  9. After you have completed your customization of the image, click [Insert] on either the General tab or the Appearance tab to insert the image into the rich text editor.
