Insert/Edit Image Dialog

This dialog opens when you insert your cursor within the content of the rich text editor or when you select a previously inserted image and click . The dialog's fields are empty or filled depending on whether you are defining a new image or editing an existing one. The dialog has a General tab and an Appearance tab.

General Tab
The General tab includes data about an inserted image, such as the Image URL and the Title. It also shows a preview of the inserted image.


Fields in General Tab
Image URL - Image URL is automatically filled with the URL of the asset you insert from CM1.
[Use Existing Image] - Clicking this button opens the following Select Image dialog, which enables you to find an image asset in CM1 and insert it into the rich text editor. See Inserting an Existing Image for instructions on using this dialog. Note: currently you can only insert an image stored in CM1 into the rich text editor.

Image description - When you insert an Image asset, Image description is filled with the text in the Alt Text field of the Image asset. You can overwrite the value of the field for this instance of the asset only.
Title - When you insert an image asset, Title is filled with the asset's title, but you can change it.
Image Size - Radio button control that lets you select one of the following two options:

  • Full Image - Inserts full-size image into text.
  • Thumbnail Image - Inserts thumbnail image into text.

Preview - Displays the image.

Appearance Tab
The Appearance tab lets you modify the format of the image.


Fields in Appearance Tab
Entries in these fields apply to this instance of the image only.
Alignment - How image is aligned within the editor. The graphic on the right of the dialog shows you how your choice will align the image, but actual alignment may vary by browser. Options are:

  • Not set - Defaults to Baseline.
  • Baseline - Bottom of graphic is lined up with bottom of text in which it is inserted.
  • Top - Top of graphic is lined up with top of text in which it is inserted.
  • Middle - Middle of graphic is lined up with middle of text in which it is inserted. Graphic spreads above and below the text.
  • Bottom - - Bottom of graphic is lined up with bottom of text in which it is inserted.
  • Text top - Top of graphic is lined up with top of text in which it is inserted.
  • Text bottom - Bottom of graphic is lined up with bottom of text in which it is inserted.
  • Left - Graphic is inserted on the left side of the editor.
  • Right - Graphic is inserted on the right side of the editor.

Dimensions - These fields are filled with the dimensions of the image after it is inserted, but you can change the values. Dimensions are measured in pixels.
Constrain proportions When checked, if you edit one of the image's Dimensions, CM1 automatically sets the other dimension in proportion.
Border - Size of a border to place around the image. The graphic on the right of the dialog shows you how wide the border's lines will be.
