Configure rxwebdav servlet for the default (tomcat) application server:

[1] Access the webdav folder by the URL:   http://host:port/rxwebdav/Sites/CoporateInvestment/Files
		                  with the alias:  http://host:port/rxwebdav/cifiles
		                  
                                     or:   http://host:port/rxwebdav/Sites/CoporateInvestment/Images
                         with the alias:   http://host:port/rxwebdav/ciimages
   
                                      or:  http://host:port/rxwebdav/Sites/EnterpriseInvestment/Files
		                  with the alias:  http://host:port/rxwebdav/eifiles
		                  
                                     or:   http://host:port/rxwebdav/Sites/EnterpriseInvestment/Images
                         with the alias:   http://host:port/rxwebdav/eiimages
   

    get the config file by:                http://host:port/rxwebdav/Sites/CoporateInvestment/Files?sys_command=getconfig
    validates the config file by:          http://host:port/rxwebdav/Sites/CoporateInvestment/Files?sys_command=validateconfig
